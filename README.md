# Simon Game

This is a simple Simon Game made using HTML, CSS, and JavaScript.

## Features
- Interactive color buttons
- Memory-based gameplay
- Sound effects

## How to Play
- Click the buttons in the same order as the game shows.
- The sequence gets longer each level.
